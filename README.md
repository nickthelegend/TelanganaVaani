**TelanganaVaani** is a state-of-the-art AI text-to-speech engine specifically engineered to capture the unique cadence, emotion, and cultural nuances of the Telangana dialect. Unlike generic TTS models, TelanganaVaani-v1 understands the "soul" of the language—from the warm "Bava" check-ins to the iconic "Baigan" slang.

---

## 🌟 Key Features

### 1. Cultural Awareness
The model is fine-tuned to recognize and properly enunciate regional slang and emotional markers that define the Telangana identity. It doesn't just speak Telugu; it speaks *Telangana*.

### 2. Dual-Script Support
- **Native Telugu Script**: Full support for Unicode Telugu characters.
- **Romanized Telugu**: Advanced phonetic mapping for those who prefer typing in English script (e.g., *"Em bava? Etlunnave?"*).

### 3. Real-Time Emotion
Optimized for low-latency generation with high-fidelity output, suitable for content creators, interactive agents, and cultural preservation projects.

### 4. Zero-Config Frontend
A premium, glassmorphism-inspired web interface is included for immediate deployment and testing.

## 🚀 Getting Started

To launch the TelanganaVaani hub:

1.  Navigate to the `front/` directory.
2.  Open `telangana-vaani.html` in any modern web browser.
3.  Choose a test pill (e.g., **"Bava Check-in 🤙"**) or type your own dialect text.
4.  Click **🎙️ GENERATE TELANGANA VOICE**.

---

## 🧪 Testing the Model

For standardized testing of dialect accuracy, refer to the [Test Suite](tests/romanized_tests.md). It contains specific phrases designed to push the model's understanding of:
- **Phonetic Emphasis** (e.g., the long 'aa' in *Aasha*)
- **Slang Pronunciation** (e.g., the specific 'v' sound in *Vaani*)
- **Emotional Flow** (e.g., the rising intonation in *Etlunnave?*)
